## Middleware Order (Critical!)

```
1. CORS              вҶҗ Security first
2. Compression       вҶҗ Compress early
3. JSON Parser       вҶҗ Parse body
4. Session           вҶҗ Auth setup
5. Passport          вҶҗ OAuth
6. Request Logger    вҶҗ Log everything
7. ETag (per route)  вҶҗ Cache GET requests
8. Routes            вҶҗ Business logic
9. Error Logger      вҶҗ Error handling
10. 404 Handler      вҶҗ Catch-all
```
